HTTP状态码详解

HTTP状态码（HTTP Status Code）是用以表示网页服务器HTTP响应状态的3位数字代码。它由 RFC 2616 规范定义的，并得到RFC 2518、RFC 2817、RFC 2295、RFC 2774、RFC 4918等规范扩展。
当我们从客户端向服务端发送请求时，服务器向我们返回HTTP状态码。
HTTP状态码就是告诉我们服务器响应的状态，通过返回的HTTP状态码，我们可以清楚知道当前请求的状态。

消息(1XX)

1XX-信息性状态码

成功(2XX)

2XX-成功状态码

重定向(3XX)

3XX-重定向状态码

客户端请求错误(4XX)

4XX-客户端错误状态码

服务器错误(5XX)

5XX-服务器错误状态码
